2. html head body의 역할 ?
<html 문서의 시작, 끝을 표시해주는 태그>
<head></head> : 머리부 => 해당 문서의 각종 정보와 제목, 설명 및 스크립트, 스타일 시트 내용들 작성 <meta>,<title>,<link>
<body></body> : 몸체부 => 화면상에 출력해서 보여주고자 하는 모든 정보/내용 들을 작성하는 부분

3. h 태그 (종류, 설명)
<h1></h1>~<h6></h6>
제목태그 숫자가 커질수록 글씨는 작아진다.

4. html의 기본 구조
<!DOCTYPE html> : html 문서 형식 선언 : html5형식이란걸 알려주기 위한 태그
<html> : 문서의 시작
<head></head> : 머리부
<body></body> : 몸체부
</html>

5. 각종태그들.. br, p, pre, a, img(src alt), ul, ol(로마자)
br : 줄바꿈태그

p, pre: 문단을 나누는태그

p와 pre의 차이점 : p태그는 줄바꿈을 할때 별도의 태그를 작성해야하고 여러개의 공백은 하나의 공백으로 인식
		pre태그는 시작태그에서부터 종료태그까지 입력된 내용을 그대로 표현하는 태그이다. 줄바꿈이라거나 여러개의 공백을 바로 인식한다.

a : 하이퍼링크 태그 a태그의 href 속성으로 링크를 만들 수 있다.

img(src alt) : 이미지 태그
img src = 삽입하고자 하는 이미지의 경로
alt : 사진의 경로가 잘못되었거나 이미지를 제대로 표현할 수 없는 경우 대체 텍스트의 용도
      시각장애인들을 위한 스크린리더에서 이미지를 읽어주는 설명 문구

ul : 순서가 없는 목록태그
ol : 순서가 있는 목록태그

6. 블럭속성 인라인 속성
블럭요소 : 한 줄 단위로 영역을 차지한다.
	줄바꿈이 적용되어 이미 존재하는 태그의 다음줄에 영역이 잡힌다.
 	(p, pre, "div", ...)
인라인요소 : content 영역에 해당하는 부분만 영역을 차지하는 요소
	   줄바꿈이 적용되지 않아 옆으로 영역이 잡힌다.
	  (b, mark, s, img, "span", ... )

7. 표태그 병합
셀(th,td)태그의 속성을 이용해서 행 또는 열을 합칠 수 있다.
colspan="2" => 2개의 열을 합친다.
rowspan="2" => 2개의 행을 합친다.

8. 아이디랑 클래스의 차이점
클래스는 중복 사용이 가능하지만 아이디는 페이지에서 한번만 사용가능하다.
클래스는 한 요소에 여러개의 클래스명이 적용될수 있고, 아이디는 한요소에 한 아이디만 적용이 가능하다.

9. input type = ?? // 타입 속성을 알아야 한다.
input은 인라인요소
텍스트상자 또는 체크박스 등등을 만들수 있다.
text, password, radio, submit, search, url, email, phone, number
radio 라디오 버튼은 name속성값이 똑같은 것들끼리 하나의 그룹으로 지어진다.
